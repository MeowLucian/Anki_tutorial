## 插入聲音：

* 自行加入

安裝擴充功能

使用 Chrome 瀏覽器搜尋 [Chrome 線上應用程式商店](https://chrome.google.com/webstore/category/extensions)

在左上角搜尋「下載網路字典發音」

點選製作者為 ```Chien Chun``` 的擴充功能

複製聲音檔

```
快捷鍵 滑鼠右鍵 + E
```

* AwesomeTTS

```
快捷鍵 Ctrl + T
```
